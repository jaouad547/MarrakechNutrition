# Task 01: Project Setup

## Goal
Initialize a new Laravel 11 project with Inertia.js, React, Tailwind CSS, and all required dependencies.

## Parent Milestone
[Milestone 1: Project Foundation](../00-main-plan.md)

## Prerequisites
None

## Acceptance Criteria
- [ ] Laravel 11 project is installed and the welcome page loads without errors
- [ ] Inertia.js middleware is registered and configured
- [ ] React root component mounts and renders on the home page
- [ ] Tailwind CSS classes are applied correctly and utility styles are visible
- [ ] Vite development server starts without errors
- [ ] Project environment file is configured with database credentials
- [ ] Application key is generated
- [ ] The `/` route renders an Inertia-powered React page instead of the default Blade view

## Files to Create / Modify
- `.env`
- `composer.json`
- `package.json`
- `vite.config.js`
- `resources/js/app.jsx`
- `resources/views/app.blade.php`
- `routes/web.php`
- `app/Http/Kernel.php`

## Estimated Duration
2–3 hours

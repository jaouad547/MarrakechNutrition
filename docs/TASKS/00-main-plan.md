# MarrakechNutrition — Main Roadmap

## Project Overview

| Property | Value |
|----------|-------|
| **Name** | MarrakechNutrition |
| **Type** | E-commerce website for nutritional supplements |
| **Target Audience** | Fitness enthusiasts and health-conscious residents of Marrakech |
| **Stack** | Laravel 11 + Inertia.js + React + MySQL + Tailwind CSS |
| **Payment** | Cash on delivery (primary), online payment (simulated) |

## Design Resources

All visual design specifications, color palettes, typography rules, and component guidelines are provided in the `/DESIGN/` folder. This includes the complete design system and design philosophy for the project. No additional design files need to be created.

# Task 20: Testing & Deployment

## Goal
Verify all features work correctly through manual testing and prepare the application for production deployment.

## Parent Milestone
[Milestone 8: Polish & Deploy](../00-main-plan.md)

## Prerequisites
- All previous tasks

## Acceptance Criteria
- [ ] All navigation links work correctly across the site
- [ ] User registration, login, and logout function without errors
- [ ] Products can be added to cart with correct quantity and price
- [ ] Cart quantities can be updated and items can be removed
- [ ] Complete checkout flow works from cart to order confirmation
- [ ] Order numbers are generated uniquely for each order
- [ ] Stock decreases correctly after order placement
- [ ] Admin can create, edit, and delete products
- [ ] Admin can update order statuses and cancel orders
- [ ] Client dashboard shows correct order history
- [ ] Role-based access prevents non-admin users from accessing admin routes
- [ ] Application runs without errors in production environment
- [ ] Environment is configured for production (debug off, app URL set)

## Files to Create / Modify
- Environment configuration file
- Production build scripts
- Deployment checklist

# Task 18: Responsive Design

## Goal
Ensure all pages display and function correctly across mobile, tablet, and desktop screen sizes.

## Parent Milestone
[Milestone 8: Polish & Deploy](../00-main-plan.md)

## Prerequisites
- All previous tasks

## Acceptance Criteria
- [ ] Home page layout adapts from single column on mobile to multi-column on desktop
- [ ] Product grid shows 1 column on mobile, 2 on tablet, 3-4 on desktop
- [ ] Navigation collapses into a hamburger menu on screens below tablet width
- [ ] Cart page table converts to a card-based layout on mobile
- [ ] Checkout form stacks vertically on mobile and uses side-by-side layout on desktop
- [ ] Admin dashboard sidebar becomes a collapsible drawer on mobile
- [ ] All tables in admin panels support horizontal scroll on mobile
- [ ] Touch targets are at least 44x44 pixels on mobile
- [ ] No horizontal scrolling occurs on any page at any screen width
- [ ] Text remains readable without zooming at all screen sizes

## Files to Create / Modify
- All page components for responsive adjustments
- Navigation component for mobile menu
- Admin layout for responsive sidebar
- CSS/Tailwind responsive utility classes

## Estimated Duration
4–5 hours

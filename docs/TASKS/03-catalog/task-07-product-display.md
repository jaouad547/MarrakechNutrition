# Task 07: Product Display

## Goal
Display products to visitors with listing pages, filtering, search, and detailed product views.

## Parent Milestone
[Milestone 3: Product Catalog](../00-main-plan.md)

## Prerequisites
- [Task 06: Products CRUD](../03-catalog/task-06-products-crud.md)

## Acceptance Criteria
- [ ] Home page displays featured products in a grid layout
- [ ] Products listing page shows active products with image, name, price, and category
- [ ] Products are paginated with 12 items per page
- [ ] Visitors can filter products by category using a sidebar or dropdown
- [ ] Visitors can filter products by price range (min/max)
- [ ] Visitors can search products by name using a search field
- [ ] Only active products appear on the frontend
- [ ] Product detail page displays large image, name, price, description, stock status, and add-to-cart button
- [ ] Product detail page shows related products from the same category
- [ ] URLs use product and category slugs for readability

## Files to Create / Modify
- Home page component
- Product listing page component
- Product detail page component
- Product card component
- Product filtering component
- Search component
- Frontend product controller
- Routes for product display

4–5 hours
